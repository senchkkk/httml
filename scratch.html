Scratch

Description

Scratch is a visually blocky event-oriented programming environment created for children and teenagers. The name comes from the word scratching, a technique used by hip—hop DJs who spin vinyl records back and forth with their hands in order to mix musical themes. The Scratch cat is an official character in the Scratch programming language, with which the entire history of the language is associated. Programming

The main components of the scratch program are sprite objects. A sprite consists of a graphical representation — a set of costume frames, a set of sounds, and a script script. A graphic editor is built into scratch to edit sprite costumes. The action of the Scratch program takes place on a 480×360 pixel stage with the coordinate center in the middle of the scene. The scene has its own set of backgrounds, sounds and can execute its own scripts. To program scripts in Scratch, a drag-and-drop approach is used: blocks from the block palette are dragged into the script area. According to their functional purpose, the blocks are divided into 8 groups, the belonging of the block to one or another group is indicated by its color. Modifications and further development

Based on the Scratch 1.4 source code, a number of language modifications have been created, such as: 1.BYOB 2.Panther 3.Slash Main article: Snap! Developed at the University of Berkeley. The main extension of the language, which was introduced in BYOB, was the ability to build composite user blocks — an analogue of the procedures of conventional programming languages. Recursion, closures, and lambda expressions are supported. A debugger and the ability to compile into executable files, nested sprites, multidimensional lists, improved scrolling and compilation of executable files have also been added. Since version 3.1, BYOB has added support for OOP — BYOB sprites now allow prototype-based inheritance. To do this, the sprite cloning mechanism has been introduced into the language, in particular.

Description

Java is a strongly typed general-purpose object-oriented programming language developed by Sun Microsystems. Development is conducted by a community organized through the Java Community Process; the language and the main technologies implementing it are distributed under the GPL license. Java

History of creation

Initially, the language was called Oak, developed by James Gosling for programming household electronic devices. Due to the fact that a language with that name already existed, Oak was renamed Java. It is named after the Java coffee brand, which, in turn, received the name of the island of the same name, therefore, a cup with hot coffee is depicted on the official emblem of the language. There is another version of the origin of the name of the language, associated with an allusion to a coffee machine as an example of a household device for programming which the language was originally created. In accordance with etymology in Russian-language literature from the end of the twentieth century to the first years of the twenty-first century, the name of the language was often translated as Java, rather than transcribed. Application

The following successful projects have been implemented using Java (J2EE) technologies: RuneScape, Amazon, eBay, LinkedIn, Yahoo!. The following companies mainly focus on Java (J2EE) technologies: SAP, IBM, Oracle. In particular, the Oracle Database includes the JVM as its component part, which provides the possibility of direct programming of the database in the Java language, including, for example, stored procedures.

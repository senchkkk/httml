TypeScript

Description

TypeScript is a programming language introduced by Microsoft in 2012 and positioned as a web application development tool that extends the capabilities of JavaScript. The developer of the TypeScript language is Anders Heilsberg, who previously created Turbo Pascal, Delphi and C#. The language specifications are open and published under the Open Web Foundation Specification Agreement. TypeScript is backward compatible with JavaScript and compiles to the latter. In fact, after compilation, a TypeScript program can be executed in any modern browser or used in conjunction with a server platform Node.js . The code of an experimental compiler that translates TypeScript into JavaScript is distributed under the Apache license. It is being developed in a public repository via the GitHub service. TypeScript

Refence

TypeScript originated due to the perceived shortcomings of JavaScript in large-scale applications both at Microsoft and other JavaScript users. Problems with the development of complex JavaScript programs have led to the need to facilitate the development of language components. TypeScript developers were looking for a solution that would not violate compatibility with the standard and its cross-platform support. Knowing that only the ECMAScript standard offers future support for Class-based programming, they based TypeScript on this assumption. This led to the creation of a JavaScript compiler with a set of syntactic language extensions, enlarged based on a sentence that transforms extensions into JavaScript. In this sense, TypeScript is a representation of what to expect from ECMAScript 6.

PHP

Description

PHP is a general—purpose scripting language that is intensively used for developing web applications. Currently, it is supported by the vast majority of hosting providers and is one of the leaders among the languages used to create dynamic websites. The language and its interpreter are being developed by a group of enthusiasts as part of an open source project. The project is distributed under its own license, incompatible with the GNU GPL. The syntax of PHP

The syntax of PHP is similar to the syntax of the C language. Some elements, such as associative arrays and the foreach loop, are borrowed from Perl. The lexical analysis of PHP is carried out using the re2c utility, the utility generates fast and easily embedded lexers. To write a simple script, you do not need to describe any variables, modules used, etc. Any script can start directly with the . The closing statement is optional and it is recommended to omit it to avoid problems with sending extra characters to the output stream after executing the main code. The PHP Standard Guidelines

The PHP Standard Guidelines are a PHP specification that is developed, accepted and published by a group of reputable PHP projects including CakePHP, Composer, Joomla, PEAR, Phalcon, phpDocumentor, TYPO3, Yii framework. The specification serves as a standardization of programming concepts in the PHP language and is based on real-world experience, as well as research and experiments. The purpose of the recommendations is to ensure compatibility of web application components developed by different developers. (the information is taken from the website using the button below)
